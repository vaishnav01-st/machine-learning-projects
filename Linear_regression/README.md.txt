# Instagram Likes Prediction Model

##  Problem Statement
Predict how many likes an Instagram post will get based on:
- Followers count
- Number of hashtags
- Caption length
- Posting time

##  Dataset
Synthetic dataset with 100 records.

##  Model Used
- Linear Regression
- StandardScaler for feature scaling

## Model Performance
- R2 Score: 55%
- MAE: 339
- Accuracy (Mean % based): 71%

## Technologies Used
- Python
- Pandas
- NumPy
- Scikit-learn
- Matplotlib

##  Sample Prediction
For:
- 12,000 followers
- 23 hashtags
- 60 caption length
- 5 AM posting time

Predicted Likes: ~1980
